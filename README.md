# Simple TypeScript Server

## Prerequisite

```bash
# bash
npm install -g yarn
```

## Installation

```bash
# bash
yarn
```

## Run Development

```bash
# bash
yarn dev
```

## Build Production

```bash
# bash
yarn build
```

## Run Production

```bash
# bash
yarn start
```
